# 月度积分自动发放方案

## 🎯 目标

实现 `grantMonthlyFreeCredits` 函数的自动调用，每月为免费用户发放积分。

## 🛠️ 实现方案

### 方案1: Vercel Cron Jobs（推荐）

**适用场景**: 部署在 Vercel 平台
**优点**: 简单、可靠、无需额外服务
**缺点**: 仅限 Vercel 平台

#### 实现步骤

1. **在 `vercel.json` 中配置 Cron Jobs**:
```json
{
  "crons": [
    {
      "path": "/api/cron/monthly-credits",
      "schedule": "0 2 1 * *"
    }
  ]
}
```

2. **Cron 表达式说明**:
   - `0 2 1 * *`: 每月1号凌晨2点执行
   - `0 0 1 * *`: 每月1号午夜执行
   - `0 10 1 * *`: 每月1号上午10点执行

3. **API 端点已创建**: `src/app/api/cron/monthly-credits/route.ts`

#### 部署配置
```bash
# 1. 更新 vercel.json
echo '{
  "crons": [
    {
      "path": "/api/cron/monthly-credits",
      "schedule": "0 2 1 * *"
    }
  ]
}' > vercel.json

# 2. 部署到 Vercel
vercel --prod
```

### 方案2: GitHub Actions（通用方案）

**适用场景**: 任何平台部署
**优点**: 免费、灵活、平台无关
**缺点**: 需要配置 GitHub Actions

#### 实现步骤

1. **创建 GitHub Action 工作流**:
```yaml
# .github/workflows/monthly-credits.yml
name: Monthly Credits Distribution

on:
  schedule:
    # 每月1号凌晨2点 (UTC时间)
    - cron: '0 2 1 * *'
  workflow_dispatch: # 允许手动触发

jobs:
  grant-monthly-credits:
    runs-on: ubuntu-latest
    steps:
      - name: Grant Monthly Credits
        run: |
          curl -X POST \
            -H "Content-Type: application/json" \
            -H "Authorization: Bearer ${{ secrets.CRON_SECRET }}" \
            "${{ secrets.APP_URL }}/api/cron/monthly-credits"
```

2. **在 GitHub Secrets 中配置**:
   - `APP_URL`: 你的应用URL
   - `CRON_SECRET`: 用于验证的密钥

3. **更新 API 端点以支持认证**:
```typescript
// src/app/api/cron/monthly-credits/route.ts
export async function GET(request: Request) {
  // 验证请求来源
  const authHeader = request.headers.get('authorization');
  const expectedAuth = `Bearer ${process.env.CRON_SECRET}`;
  
  if (authHeader !== expectedAuth) {
    return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
  }
  
  // ... 执行积分发放逻辑
}
```

### 方案3: 云函数定时触发

#### 3.1 使用 Cloudflare Workers Cron

**适用场景**: 使用 Cloudflare
**优点**: 全球分布、高可用
**缺点**: 需要 Cloudflare Workers 配置

```javascript
// wrangler.toml
[triggers]
crons = ["0 2 1 * *"]

// worker.js
addEventListener("scheduled", event => {
  event.waitUntil(handleScheduledEvent(event));
});

async function handleScheduledEvent(event) {
  const response = await fetch(`${APP_URL}/api/cron/monthly-credits`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${CRON_SECRET}`
    }
  });
  
  console.log('Monthly credits response:', await response.json());
}
```

#### 3.2 使用 AWS Lambda + EventBridge

**适用场景**: AWS 生态系统
**优点**: 高度可配置、企业级
**缺点**: 配置复杂、可能有成本

```javascript
// lambda function
exports.handler = async (event) => {
  const response = await fetch(`${process.env.APP_URL}/api/cron/monthly-credits`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.CRON_SECRET}`
    }

  return {
    statusCode: 200,
    body: JSON.stringify(await response.json())
  };
};
```

### 方案5: 应用内部定时器（不推荐）

**适用场景**: 开发测试
**优点**: 简单直接
**缺点**: 不可靠、单点故障

```typescript
// src/lib/scheduler.ts
import { grantMonthlyFreeCredits } from '@/server/cron/monthly-credits';

class CronScheduler {
  private intervalId: NodeJS.Timeout | null = null;

  start() {
    // 每小时检查一次是否需要执行
    this.intervalId = setInterval(async () => {
      const now = new Date();
      
      // 每月1号凌晨2点执行
      if (now.getDate() === 1 && now.getHours() === 2) {
        console.log('Executing monthly credits...');
        await grantMonthlyFreeCredits();
      }
    }, 60 * 60 * 1000); // 每小时检查
  }

  stop() {
    if (this.intervalId) {
      clearInterval(this.intervalId);
      this.intervalId = null;
    }
  }
}

export const cronScheduler = new CronScheduler();
```

```typescript
export async function GET(request: Request) {
  // 验证请求来源（可选）
  const userAgent = request.headers.get('user-agent');
  if (userAgent && !userAgent.includes('Vercel-Cron')) {
    console.warn('Suspicious cron request from:', userAgent);
  }
  
  try {
    const result = await grantMonthlyFreeCredits();
    
    // 记录执行结果
    console.log('Monthly credits cron executed:', result);
    
    return NextResponse.json({
      success: true,
      message: 'Monthly credits distributed successfully',

      executedAt: new Date().toISOString(),

  } catch (error) {
    console.error('Monthly credits cron failed:', error);
    
    return NextResponse.json({
      success: false,
      message: 'Monthly credits distribution failed',
      error: error instanceof Error ? error.message : 'Unknown error',
      executedAt: new Date().toISOString(),
    }, { status: 500 });
  }
}
```

```typescript
// scripts/verify-monthly-credits.ts
async function verifyMonthlyExecution() {
  const currentMonth = new Date().toISOString().slice(0, 7);
  
  // 检查本月是否有积分发放记录
  const monthlyTransactions = await db
    .select()
    .from(creditTransactions)
    .where(
      and(
        eq(creditTransactions.source, 'subscription'),
        like(creditTransactions.referenceId, `free_${currentMonth}%`)
      )
    );
  
  console.log(`Found ${monthlyTransactions.length} monthly credit transactions for ${currentMonth}`);
  
  return monthlyTransactions.length > 0;
}
```
